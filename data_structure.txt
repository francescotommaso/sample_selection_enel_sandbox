Inputs (required as data source):

Hourly measurements (january to april) of advanced meters. CSV files (121). Source: ENEL
Extract the file 'recent_measurements.zip', which is on Google Drive, and place it on data dir.
The data structure should be:
data/recent_measurements/{csv_file}

Monthly consumption (2023) for consumers with advanced meters and atypical lists. CSV file. Source: ENEL.
Extract the file 'complementary.zip', which is on Google Drive, and place it on data dir.
The data structure should be:
data/complementary/Base_SM.csv
data/complementary/atipicos.xlsx
data/complementary/baixa_renda.xlsx

BDGD data (shapefiles) (for 2023) with data for consumption for all consumers in ENEL. Source: ENEL.
Extract the file 'UCBT.zip', which is on Google Drive, and place it on data dir.
The data structure should be:
data/UCBT/{shapefile}

Census data for SP (2010). Source: IBGE.
Extract the file 'SP_Capital_20231030.zip', 'SP_Exceto_Capital_20231030.zip' and 'sp_setores_censitarios_2010.zip', 
which are on Google Drive, and place it on data dir.
The data structure should be:
data/sp_setores_censitarios_2010/{rest of data}
data/SP_Capital_20231030/{rest of data}
data/SP_Exceto_Capital_20231030/{rest of data}


Outputs (directories structure required for intermediary and final outputs). Make sure the following, empty directories, exist:

Meters to integers and integers to meters dictionaries (for HDF5 reference):
Origin -> hdf5_mappings.py
data/misc/

Hourly measurements clean:
Origin -> clean_csv_data.py
data/recent_measurements_clean/

HDF5, memory efficient data:
data/hdf5/

Consolidated consumption data:
Origin -> data_consolidation.py
data/consolidated/

Samples results:
Origin -> sample_selection.py
data/samples/group_1/
data/samples/group_2/
data/samples/group_3/
data/samples/group_4/
data/samples/group_5/
data/samples/merged/
